#  Raport Zarobkowy Data Acolyte 2025

Analiza zarobków w branży danych w Polsce na podstawie anonimowej ankiety społeczności Data Acolyte.

## Cel projektu
Transparentny obraz rynku pracy w obszarze analityki danych, BI i data engineering w Polsce.

## Struktura projektu
- `data/` - dane surowe i przetworzone
- `notebooks/` - notebooki Jupyter z analizą
- `outputs/` - wykresy i finalne materiały
- `scripts/` - skrypty pomocnicze

## Setup
```bash
pip install -r requirements.txt
```

## Metodologia
- Próba: n=40 respondentów
- Źródło: społeczność Data Acolyte
- Okres: 2025
- Typ: ankieta samoopisowa (bias: self-reported data)
